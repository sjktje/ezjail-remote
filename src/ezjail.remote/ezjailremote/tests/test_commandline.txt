Basic Argument handling
=======================

An unknown command raises a SystemExit:

  >>> import os
  >>> from ezjailremote.command import ezjailremote
  >>> ezjailremote(args=['xxx', '-v'])
  Traceback (most recent call last):
  ...
  SystemExit: 1

as does providing more than three arguments:

  >>> ezjailremote(args=['1', '2', '3', '4'])
  Traceback (most recent call last):
  ...
  SystemExit: 1

Providing no command shows the usage:

  >>> ezjailremote(args=[])
  usage: ezjailremote <command> [options] [args]
  ...

Calling the help command with a specific command name shows its help function:

  >>> ezjailremote(args=['help', 'jls'])
  Usage: test host jls
  ...

Finally, let's actually call a command:

  >>> ezjailremote(args=['jls', '-v'])
